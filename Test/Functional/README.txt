To run this tests

- move this folder into  `dev/tests/functional/tests/app/MX/HelperBar/Test`
- add "MX\\": "tests/app/MX/" into the dev/tests/functional/composer.json psr-4 section

Also make sure you have configured Magento to run Functional tests as specified on the docs:
http://devdocs.magento.com/guides/v2.0/mtf/mtf_quickstart.html
